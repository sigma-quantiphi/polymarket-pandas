# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and help others learn. Harassment, discrimination, and personal attacks are not tolerated.

## Enforcement

Report unacceptable behavior privately via [GitHub's private vulnerability reporting](https://github.com/sigma-quantiphi/polymarket-pandas/security/advisories/new) or directly to the maintainers. Reports will be reviewed confidentially.

Consequences range from a private warning to a permanent ban, proportional to the severity and pattern of the behavior.
